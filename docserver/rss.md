
# rss verbs
## rss.readFeed
#### Syntax
rss.readFeed (url)

#### Params
The url is the address of an RSS feed. 

#### Returns
A JavaScript object with the contents of the feed.

#### Note
It can handle most versions of RSS and Atom. 

#### Example
rss.readFeed ("http://scripting.com/rss.xml")
